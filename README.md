# Flex Full Flow Review

Static share package for the Flex Full Flow Review prototype.

## Open locally

Open `flex-discounts-review/index.html` in Chrome or Edge.

## GitHub Pages

This repo is designed to be published from the repository root with GitHub Pages.
The root `index.html` redirects to:

```text
./flex-discounts-review/
```

## Contents

- `PM-EXPORT.md` - product handoff
- `flex-discounts-review/` - prototype HTML, CSS, and mocked data
- `_shared/` - shared assets, fonts, tokens, and components

All data is mocked. Browser events stay in localStorage.
